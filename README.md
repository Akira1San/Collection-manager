# Collection Manager

A media collection manager for the **AkiraTV IP Home Streamer**. This is a subsystem that manages, organizes, and catalogs video collections with metadata fetching, tag management, and cover art support.

## Features

- Scan and catalog video files
- Fetch metadata from online sources
- Manage tags and categories
- Cover art display
- Collection organization

## Requirements

- Python 3.x
- PySide6

## Usage

Run the application:

```bash
./run.sh
# or
python main.py
```

## Project Structure

- `main.py` - Entry point
- `src/` - Application source code
- `run.sh` - Launch script
